# 隐私权政策 (Privacy Policy)

**最近更新日期：** 2026年6月18日

**Input Translation**（以下简称“本插件”）是一款开源的浏览器扩展，旨在帮助用户直接在输入框、网页选中文本以及地址栏中进行文本翻译。我们非常重视您的隐私，并承诺保护您在使用本插件时处理的任何信息。

本隐私权政策旨在向您说明本插件处理哪些信息、如何使用这些信息以及您对这些信息的控制权。

---

## 1. 开发者不收集任何数据

**我们不收集、不获取、不存储、也不向任何第三方出售您的个人数据。**

与许多商业服务不同，本插件不需要用户注册或创建账户。开发者不运营任何用于收集、分析或汇总用户数据的中心化服务器。所有的翻译操作均在您的本地设备上执行，或直接发送至经您授权的第三方翻译 API 服务。

---

## 2. 处理哪些信息以及如何使用

为了实现核心的翻译功能，本插件会与以下数据进行交互：

### A. 提交翻译的文本（用户内容）
*   **处理内容：** 您在网页上高亮/选中的文本、在地址栏中输入的翻译内容（带翻译指令）、以及您在任何输入框中输入的文本（以翻译代码如 `/zh` 结尾）。
*   **处理方式：** 这些文本会在浏览器内存中进行临时处理，并通过安全的 HTTPS 协议直接加密传输至：
    1.  **百度翻译官方 API** (`https://api.fanyi.baidu.com`)。
    2.  如果您配置了自定义代理，则传输至您**自主托管的代理服务器**（例如 Cloudflare Workers）。
*   **保存期限：** 我们不会保存或记录您翻译的任何文本。一旦翻译结果返回并插入您的输入框或显示在屏幕上，该文本就会从本插件的活动内存中清除（为了避免重复的网络请求，本插件可能在内存中进行极其短暂的会话缓存，这些缓存会在您关闭标签页或浏览器时完全清空）。

### B. 插件设置和配置
*   **处理内容：** 您的个性化偏好设置，例如默认目标语言、划词翻译开关、以及您配置的任何自定义 API 密钥或代理 URL。
*   **处理方式：** 这些配置将直接保存在您浏览器的本地同步存储中（`chrome.storage.sync`）。
*   **保存期限：** 这些数据保存在您的本地机器上，并可能通过您浏览器的内置同步服务（如 Microsoft 账户同步或 Google 账户同步）在您的不同设备之间同步。开发者无法访问这些存储内容。

---

## 3. 请求的权限及其必要性说明

为了实现相应的功能，本插件在浏览器清单（`manifest.json`）中请求了以下特定权限：

*   **`storage`（存储）：** 用于保存您的插件配置和偏好设置（如目标语言、功能开关、API 密钥等），免去您每次打开浏览器都要重新配置的麻烦。
*   **`activeTab` 和 `tabs`（标签页）：** 用于检测您何时在当前活动的浏览器标签页中选中了文本或聚焦于输入框，以便本插件能够渲染翻译泡泡并插入翻译后的文本。
*   **`contextMenus`（右键菜单）：** 用于在您浏览器的右键快捷菜单中添加“翻译选中文本”的快捷选项。
*   **`host_permissions`（主机权限 - `<all_urls>`、`https://api.fanyi.baidu.com/*`）：** 用于在网页上运行内容脚本，以便您可以在任意网站（如 ChatGPT、Claude、GitHub 等）中进行行内翻译，并授权本插件与百度翻译 API 或您自定义的 Cloudflare Worker 代理进行网络通信。

---

## 4. 第三方服务

当您发起翻译请求时，选中的文本将被传输到第三方服务以生成翻译结果。根据您的配置，这些服务可能包括：

*   **百度翻译 API：** 翻译请求由百度处理。我们建议您阅读 [百度隐私政策](https://fanyi.baidu.com/) 以了解他们如何处理和保护您的数据。
*   **您部署的代理（Cloudflare Workers）：** 如果您通过自定义的 Cloudflare Worker 代理来转发请求，数据传输和处理将受到 Cloudflare 的隐私规范以及您个人配置的约束。

---

## 5. 数据安全

我们非常重视您的数据安全。本插件与翻译 API 之间的所有通信均使用行业标准的 **HTTPS** 协议进行传输加密。

---

## 6. 儿童隐私保护

本插件不针对也无意吸引或收集儿童的任何信息。由于我们完全不收集任何个人数据，因此不会存储任何儿童的数据。
